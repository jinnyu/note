## 使用体会

优点 :

1. 价格足够便宜
   因为家里有私有服务器, 所以公有云的需求就只剩带宽了, 所以越便宜越好.
2. 管理后台交互体验好
   滴滴云的管理后台是我用过这么多后台当中体验最好的.
3. 工单响应及时

缺点 :

1. 主机监控插件
   监控插件占用CPU略高, 1C配置下占用率达到10%左右.
2. 可用区只有广州和北京
   * 2021-09-06 目前只有广州可用, 北京售罄.
